EasyRead Privacy Policy

EasyRead does not collect or store personal data.

When a user selects text and requests an explanation, the selected text is sent to the EasyRead server and processed using OpenAI APIs to generate a simplified explanation.

No browsing history, personal identity information, or persistent user data is stored.

The data is used only to generate the explanation and is not sold or shared with third parties.
